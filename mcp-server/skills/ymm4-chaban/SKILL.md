# ゆっくり茶番スキル

霊夢（ボケ・天然）と魔理沙（ツッコミ・しっかり者）の掛け合いで
笑いを作る茶番劇を自動生成するスキル。テンポとリズムが命。

---

## キャラクター性

- **霊夢（ボケ・天然役）**：マイペース・天然・ズレた言動・自覚なし
  - 口調：「〜じゃない？」「なんとなくそう思って」「え、違うの？」「普通そうでしょ」
- **魔理沙（ツッコミ・進行役）**：しっかり者・素早いリターン・呆れながらも愛ある突っ込み
  - 口調：「違うだろ！」「なんでそうなるんだ！」「普通に考えろ！」「お前だけだぞそれ！」

> ⚠️ **魔理沙の一人称は「私」。「俺」「僕」は使わないこと。**

---

## 茶番動画の構成パターン

### パターンA：日常コント型
```
[日常の1シーン設定]
霊夢がおかしなことを言う・する（ボケ）
魔理沙がツッコむ
霊夢がさらに天然ボケを重ねる（自覚なし）
魔理沙が呆れてオチをつける
```

### パターンB：漫才型
```
[テーマ設定・フリ（魔理沙）]
霊夢がズレた発言（ボケ）
魔理沙「なんでそうなるんだ！」系のツッコミ
繰り返し3〜4回
大オチ（霊夢が天然でとどめ / 魔理沙が自滅）
```

### パターンC：状況設定型
```
[あり得ない状況や設定を提示]
魔理沙が進行・説明しようとする
霊夢が天然ボケで状況をどんどん混乱させる
オチで状況がさらに悪化して笑いに
```

---

## 標準ワークフロー

### Step 1：ネタの設計

茶番はテンポが命。先に**ネタ構造**を決める。

**ネタシートの作り方：**
```
設定：[どんな場面・状況か]
テーマ：[何の話をするか]
霊夢のボケ種類：[天然/勘違い/言葉遊び/大げさ/マイペース]
オチの種類：[どんでん返し/霊夢が天然でとどめ/魔理沙が自滅/第三者介入]

構造：フリ→ボケ1→ツッコミ1→ボケ2→ツッコミ2→ボケ3→大オチ
```

**ボケの種類別テンプレ：**
```
天然ボケ：
  霊夢「[全く見当違いな解釈・行動]じゃない？」
  魔理沙「違うだろ！[正しい答え]に決まってるだろ！」

言葉遊び・勘違い：
  霊夢「[ダジャレ・語感の勘違い]ってことでしょ？」
  魔理沙「そんな意味じゃないぞ！[正しい意味]だ！」

大げさ（霊夢が些細なことを大事件のように言う）：
  霊夢「大変！[些細なこと]なのよ！どうしよう！」
  魔理沙「大げさすぎるだろ！それくらいで騒ぐな！」

逆説・ひっくり返し：
  霊夢「[一見正しそうだが実は逆のこと]なんじゃない？」
  魔理沙「どういう理論だ！逆だぞ逆！」

マイペース無視：
  魔理沙「[真剣な話をしている]」
  霊夢「[全く関係ないことを言う]」
  魔理沙「聞いてたか！？」
```

### Step 2：テンポ設計のルール

茶番はセリフを**短く・テンポよく**。

```
ボケ（霊夢）：15〜25文字（短くリズミカルに）
ツッコミ（魔理沙）：10〜20文字（素早く返す）
フリ・説明（魔理沙）：20〜35文字（長すぎない）
オチ：15〜30文字（余韻を持たせる）
```

**間隔のルール：**
- ツッコミはボケ終了の **3〜5f後** に配置（間髪入れず）
- フリとボケの間：**5〜15f**（少し間を置いて霊夢が天然発言）
- オチの後：**20〜30f** 沈黙（余韻）

### Step 3：台本作成

```
フリ（魔理沙）：[状況説明・テーマ設定]  20〜30文字
  ↓ 10f後
ボケ1（霊夢）：[天然・ズレた発言]       15〜25文字
  ↓ 3〜5f後
ツッコミ1（魔理沙）：[素早く返す]        10〜20文字
  ↓ 5f後
ボケ2（霊夢）：[さらにズレる・深める]   15〜25文字
  ↓ 3〜5f後
ツッコミ2（魔理沙）：[より強いツッコミ]  15〜20文字
  ↓ 5f後
ボケ3（霊夢）：[とどめの天然・エスカレート] 20〜25文字
  ↓ 3〜5f後
大オチ（魔理沙 or 霊夢）：[落ち・締め]  15〜25文字
```

### Step 4：add_scriptで配置

```python
# 茶番はstart_frameを細かく指定してテンポを作る
# chars_per_sec=6 で少し早口に

# フリ（魔理沙）
action="add_script", start_frame=0, fps=30, chars_per_sec=6,
lines=[{"layer": 8, "character": "ゆっくり魔理沙", "text": "フリのセリフ"}]
# → total_frames が次のstart_frame目安

# ボケ1（霊夢）フリ終了+10f後
action="add_script", start_frame=フリ終了f+10, fps=30, chars_per_sec=6,
lines=[{"layer": 7, "character": "ゆっくり霊夢", "text": "ボケ1"}]

# ツッコミ1（魔理沙）ボケ終了+3f後
action="add_script", start_frame=ボケ1終了f+3, fps=30, chars_per_sec=6,
lines=[{"layer": 8, "character": "ゆっくり魔理沙", "text": "ツッコミ1"}]

# 以降繰り返し…
```

### Step 5：watchでテンポ確認

```python
action="watch", frame=0, duration_ms=8000, capture_interval_ms=8000
# → テンポが速すぎ/遅すぎなら各セリフのフレームを微調整
```

## ツッコミ（魔理沙）バリエーション

| 種類 | 例 |
|---|---|
| 標準 | 「違うだろ！」「そうじゃないぞ！」 |
| 強め | 「なんでそうなるんだ！？」「意味わからんぞ！」 |
| 呆れ | 「もう知らん…」「お前って本当に…」 |
| 怒り | 「いい加減にしろ！」「何回言えばわかるんだ！」 |
| 感心ツッコミ | 「なんでそれで合ってるんだ！？」「天才か！？」 |

## ボケ（霊夢）テクニック

| テクニック | 説明 |
|---|---|
| 天丼 | 同じ天然ボケを3回繰り返して最後に変化 |
| 純粋な勘違い | 本当に正しいと信じてズレたことを言う |
| 的外れ | 全く関係ない方向に話を持っていく |
| マイペース無視 | 深刻な状況でも普通に別の話をする |
| 正論のような間違い | 一瞬正しそうに聞こえるが実は全然違う |

## フレーム計算早見表（茶番用：6文字/秒）

| 文字数 | フレーム | 時間 |
|---|---|---|
| 10文字 | 50f | 1.7秒 |
| 15文字 | 75f | 2.5秒 |
| 20文字 | 100f | 3.3秒 |
| 30文字 | 150f | 5秒 |
