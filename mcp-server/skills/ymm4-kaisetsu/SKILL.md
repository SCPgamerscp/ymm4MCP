# ゆっくり解説スキル

テーマを「解説役の魔理沙」が「聞き役の霊夢」に教えるスタイルで
分かりやすく解説する動画を自動生成するスキル。

---

## 解説動画の構成パターン

```
[イントロ]      テーマ発表・今日学ぶことを紹介
[導入質問]      魔理沙が「〜ってなんだ？」と素朴な疑問を投げる
[解説ブロック1] 霊夢が基礎を説明
[リアクション]  魔理沙が「なるほど！」「つまり〜ってこと？」
[解説ブロック2] 霊夢がさらに深掘り・具体例
[まとめ]        霊夢が要点を整理
[エンディング]  魔理沙が感想・霊夢が締め
```

---

## 標準ワークフロー

### Step 1：コンテンツ設計

解説動画は映像確認より**台本設計が先**。
以下を決めてからセリフを作る。

**必要な情報：**
1. テーマ（例：「スケルトンキングの攻略法」「江戸時代の歴史」）
2. 対象視聴者レベル（初心者向け？中級者向け？）
3. 解説する項目数（3〜5項目が標準）
4. 動画の長さの目安（短め2〜3分、標準5分、長め10分）

**構成メモの作り方：**
```
テーマ：[テーマ名]
ポイント1：[内容]  → 霊夢が説明、魔理沙がリアクション
ポイント2：[内容]  → 霊夢が説明、魔理沙がリアクション
ポイント3：[内容]  → 霊夢が説明、魔理沙がリアクション
まとめ：[要点]
```

### Step 2：映像と同期する場合（watchで確認）

映像付き解説（ゲーム攻略・操作説明等）の場合のみ実施。

```python
# 解説対象のシーンを確認
action="seek_capture", frame=○○  # 説明したいシーン
```

映像なし解説（テキスト+立ち絵のみ）の場合はStep 3へ。

### Step 3：台本の設計ルール

**キャラクター性：**
- 魔理沙（解説役）：自信満々・分かりやすく説明「〜なんだぜ！」「〜ってことだ！」「つまり〜ってわけだ！」
- 霊夢（生徒・聞き役）：素直・鋭い質問「〜ってどういうこと？」「なるほどね！」「つまり〜ってこと？」

> ⚠️ **魔理沙の一人称は「私」。「俺」「僕」は使わないこと。**

**セリフテンプレ：**
```
イントロ：
  魔理沙「よーし！今日は[テーマ]について解説するぜ！」
  霊夢「[テーマ]って[素朴な疑問]なの？」

解説ブロック（各ポイント）：
  魔理沙「まず[ポイント名]についてだ。[具体的な説明]なんだぜ！」
  霊夢「へぇ〜！つまり[自分なりの解釈]ってこと？」
  魔理沙「そういうことだ！[肯定or補足]ってわけだぜ！」

まとめ：
  魔理沙「じゃあ今日のポイントをまとめると、[要点1]、[要点2]、[要点3]だぜ！」
  霊夢「よく分かったわ！[感想]ね！」

エンディング：
  魔理沙「以上！[テーマ]の解説でした！ゆっくりしていってね！」
  霊夢「次回もよろしくね！」
```

### Step 4：フレーム設計

解説動画はセリフが長くなりがちなので、**間（ま）**を意識する。

```
[イントロ]     0f〜 （霊夢→魔理沙、各20〜30文字）
[ポイント1]    300f〜 （霊夢説明40〜50文字→魔理沙リアクション20文字）
[ポイント2]    800f〜 （同上）
[ポイント3]    1300f〜（同上）
[まとめ]       1800f〜（霊夢50文字、魔理沙20文字）
[エンディング] 2200f〜（霊夢→魔理沙）
```

フレーム間の余白（前のセリフ終了 + 30〜60f）を入れると
視聴者が内容を理解しやすい。

### Step 5：add_scriptで配置

```python
# セクションごとに呼ぶ
action="add_script", start_frame=0, fps=30, chars_per_sec=4,
# ※解説は少しゆっくり話す想定でchars_per_sec=4推奨
lines=[
    {"layer": 7, "character": "ゆっくり霊夢",   "text": "イントロ説明"},
    {"layer": 8, "character": "ゆっくり魔理沙", "text": "疑問・リアクション"},
]
```

### Step 6：確認・修正

```python
# 各セクションの頭フレームを確認
action="seek_capture", frame=0
action="seek_capture", frame=300
action="seek_capture", frame=800
# ...
```

## 解説セリフのコツ

| シーン | ポイント |
|---|---|
| 専門用語が出るとき | 魔理沙に「〜ってなんだ？」と質問させてから霊夢が説明 |
| 重要ポイント | 「これが一番大事なのよ！」と強調 |
| 具体例を出すとき | 「例えば〜」「たとえると〜」を使う |
| 数字・データ | 魔理沙に驚かせる「そんなに！？」 |
| まとめ | 箇条書き的に「①〜、②〜、③〜ね」 |

---

## フレーム計算早見表（解説用：4文字/秒）

| 文字数 | フレーム | 時間 |
|---|---|---|
| 20文字 | 150f | 5秒 |
| 30文字 | 225f | 7.5秒 |
| 40文字 | 300f | 10秒 |
| 50文字 | 375f | 12.5秒 |

※解説はゆっくり話す（chars_per_sec=4）想定

## レイヤー構成

```
Layer 0:  動画・背景ファイル（必要に応じて）
Layer 7:  ゆっくり霊夢（生徒・質問役）
Layer 8:  ゆっくり魔理沙（解説役・先生）
Layer 9+: ゲスト解説キャラ（必要に応じて）
```
